Sobre Portifólio

"Portifólio" é como apelidei o projeto do site onde realizo a exposição dos projetos que fiz durante os meus estudos de diversas tecnologias.

O site, feito completamente a mão, foi desenvolvido puramente com HTML5 e CSS3. Como objetivo principal deste projeto é desenvolver os conhecimentos do design de front-end para futuramente desenvolver sites e aplicações web.

Próximos Passos

Após ter aprendido sobre o funcionamento da estrutura que o HTML5 proporciona e como o CSS3 é utilizado para estilizar a página, a próxima etapa deste projeto é aprender e utilizar o Framework Bootstrap e o Angular 2.
